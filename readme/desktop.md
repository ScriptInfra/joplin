## Uninstall on Linux
 Run the following commands to uninstall Joplin Notes on Linux:
 ```shell
rm -r ~/.joplin
rm -r ~/.config/joplin-desktop
rm ~/.local/share/applications/appimagekit-joplin.desktop
```
